# Suppression Playbook — Recommended Action by Type

## P0 — Critical (same-day)

### Suppressed-Andon Cord
**What it means:** Amazon has pulled the listing due to a safety/compliance complaint or issue.
**Immediate action:**
1. Check Account Mgmt board for existing POA (Plan of Action) case
2. If no case exists: PM to open case immediately and tag Compliance
3. Compliance to draft POA addressing root cause
4. Expected resolution: 3–14 days depending on Amazon response
**ADS signal:** PAUSE immediately — no impressions possible while suppressed

### Suppressed-Product Safety Violation
**What it means:** Amazon flagged a product safety concern — often triggered by customer complaint or regulatory flag.
**Immediate action:**
1. Gather safety documentation (test reports, certifications)
2. Open case with Amazon via Seller/Vendor Central
3. Submit documentation package
4. Escalate to Guillaume if Top 10 SKU
**ADS signal:** PAUSE immediately

## P1 — High (today)

### Suppressed-Pricing Issue
**What it means:** 1P price went above MAP, or Amazon detected a price inconsistency.
**Immediate action:**
1. Check current 1P price in Vendor Central vs MAP
2. If price is above MAP: submit price correction request
3. If price looks correct: check for third-party offer causing price comparison issue
4. Cross-check Pricing team — may need manual MAP enforcement
**ADS signal:** Review — ads may still run but conversion will suffer at wrong price

### Suppressed-Reason Unknown
**What it means:** Amazon suppressed without a clear reason code — requires investigation.
**Immediate action:**
1. PM to check Seller/Vendor Central for suppression notice
2. Check listing quality dashboard for content issues
3. Look for recent category or compliance change that may have triggered
4. If no clear reason found in 24h: open Amazon case
**ADS signal:** PAUSE — wasted spend if listing not visible

### ADS Suppressed
**What it means:** The ASIN is PPC-ineligible. Campaigns are running but ads won't show.
**Immediate action:**
1. Check PPC eligibility in Seller Central (usually under advertising eligibility)
2. Common causes: listing compliance issue, category restriction, pricing violation
3. Compliance to identify and resolve root cause
4. Alert ADS Rep immediately — budget is being burned with no impressions
**ADS signal:** ALERT — campaigns running but ADS suppressed. Pause until resolved.

### DF Suppressed - High Cost / Missing Cost
**What it means:** Amazon's internal cost calculation is too high or missing, suppressing the buy box.
**Immediate action:**
1. Check COGS / landed cost in Vendor Central
2. Submit cost update or correction
3. If cost is correct, raise case with Vendor Manager
**ADS signal:** Review — may still show organically but metrics will be off

## P2 — Medium (this week)

### DF Disabled
**What it means:** Direct Fulfillment is turned off for this SKU. Customer cannot purchase via Amazon fulfillment.
**Action:**
1. Check if intentional (sometimes disabled for overstock management)
2. If unintentional: re-enable in Vendor Central → Fulfillment settings
3. If intentional: ensure 3P or FBA offer is live to cover

### Live on Few Zip Codes
**What it means:** Listing is live but only showing in limited geographic areas — usually due to FBA inventory in only one warehouse region.
**Action:**
1. Check FBA distribution — is stock only in one FC region?
2. Coordinate with Logistics to distribute to additional FCs
3. Prioritize EC/MC/WC balance for Prime coverage

### Suppressed-Hazmat Issue
**What it means:** Amazon flagged the product as containing hazardous materials requiring additional review.
**Action:**
1. Submit exemption sheet or SDS (Safety Data Sheet)
2. If incorrectly flagged: open case with documentation proving non-hazmat status

### Suppressed-IP Issue
**What it means:** A competitor has filed an IP (intellectual property) complaint.
**Action:**
1. Escalate to legal/brand registry team immediately
2. Do not attempt to reinstate without legal review
3. Gather brand registry documentation

### Suppressed-Adult Issue
**What it means:** Amazon has classified the product as adult content.
**Action:**
1. Review listing images and copy for any potentially flagged content
2. Submit case with clarification that product is not adult content
3. Remove any images that could be interpreted ambiguously

### Inactive-Low Stock / Temporary OOS
**What it means:** FBA stock hit zero, Amazon made listing inactive.
**Action:**
1. Check warehouse stock — ship to FBA urgently
2. Coordinate with Logistics on shipment timeline
3. If no warehouse stock: check PO status and ETA
